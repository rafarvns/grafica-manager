# Gráfica Manager UI Kit

Interactive prototype of the Arte Papel Gráfica Manager desktop application.

## Screens included

- **Dashboard** — KPIs, sparkline chart, donut chart, top customers table, period selector
- **Pedidos** — Order list with search/filter, badges, new order modal, click-through to order detail
- **Order Detail** — Tabs (Detalhes / Arquivos / Impressões), material config, file upload zone
- **Impressões** — Print history log with summary KPIs
- Placeholder stubs for: Clientes, Relatórios, Shopee, Configurações

## Components in Shared.jsx

`Button`, `Badge`, `KpiCard`, `Card`, `Input`, `Select`, `Sidebar`, `Header`, `Toast`, `Modal`
